# traversals

[![Coveralls Status][coveralls-image]][coveralls-url]

> Small module for graph traversals, supporting DFS and BFS with niceties added for pre- and post-order, including their reverses.


## Install

```sh
npm i traversals
```

## Usage

```js
const 
    traversals = require( 'traversals' );

traversals() // true
```
